# vehicle-task-assignment-vikor

Intelligent vehicle-task assignment system using Group VIKOR and multi-criteria decision analysis in Python.

## Features
- Vehicle ranking using Group VIKOR
- Task-specific criteria weighting
- Automatic best vehicle assignment
- Excel report generation
- Visualization charts using Matplotlib

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- XlsxWriter

## Dataset
Vehicle dataset adapted from Kaggle datasets.

## Output
The system generates:
- Best assignments per task
- Full vehicle rankings
- Task weight analysis
- Q-score visualizations
